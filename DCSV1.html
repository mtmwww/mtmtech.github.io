How our solution works?



A factory machine is equipped with MtM+ DCS machine where operation data is collected through various sensors. This information is then fed to the server through a gateway where data is monitored and analyzed. The server then can trigger alarms and generate reports to either a PC or mobile device in order to improve operation efficiency.


Specification

  • Chipset:
    • MCU: STM32 MCU (Arm Cortex-M4 core)
    • Bluetooth: MtM+ M907 SiP Module
  • Memory: RAM 256KB and Flash ROM 512KB
  • Sensor: Temperature, Humidity
  • LED: Power x1, and Output x4 (programmable)
  • Connectivity Options: : BLE and Mesh (Default), Wi-Fi (optional) and LoRa (optional)
  • Physical Button: Reset x1 and Button x6 (programmable)
  • Power: DC12V /1A
  • I/O Ports:
    • GPIO: DC output 4P Euroblock x1, AC output 4P Euroblock x1
    • I2C: 2.0mm 4P x1
    • UART: USB type-B x1
    • RS232/RS485: 2P Euroblock x1
  • Dimension: 150(L) x 100(W) x 98(H) mm
