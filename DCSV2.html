Data Collection Station

DCS V2

DCS V2 (Data Collection Station V2)是用於智慧工廠的資料收集裝置，它擁有各種無線傳輸功能與多樣化的輸入與輸出介面

資料傳輸

BT Mesh、Wi-Fi與LoRa無線傳輸技術

資料收集

多樣化的輸入與輸出介面(GPIO/SPI/I2C/UART)用來連接各種儀表與感應器

完整的解決方案(HW/FW/SW)

協助我們戶能夠在短時間內導入並上線

DCS的架構



工廠安裝DCS與依照不同需求所需的感應器後，DCS會透過Gateway將感應器收集到的資料，持續上傳至雲端或本地伺服器進行監看與分析，並將分析報告或警訊送至工廠人員的手持裝置或電腦，來達到改善工作效率、節省成本的目標







我們的成功個案

特點

  • 晶片組
    • MCU: STM32 MCU (Arm Cortex-M4核心)
    • 藍牙:MtM+ M907 SiP模組
  • 記憶體: RAM 256KB與Flash ROM 512KB
  • LED:電源x1、OTG x1與可編譯LED x5
  • 資料傳輸選項: BT Mesh(預設)、Wi-Fi(選配)與LoRa(選配)
  • 按鈕：Reset x1
  • 電源：DC電源輸出12VDC (±10%), 3A
  • 輸入與輸出介面
    • GPIO: ADC10Px1、PWM 4P x1與GPIO 16P x1
    • SPI: SPIx 16
    • I2C: 2.0mm 4P x1
    • UART: USB type-B x1
    • RS232/RS485: 2P歐式端子x1
  • 尺寸：150(L) x 100(W) x 55(H) mm

應用

IoT

  • 環境監看與測量
    • 溫度 / 溼度/ 煙霧/ 震動/ 音量/ 氣壓/ 空氣品質
    • 即時監測，若有任何數值接近危險值會立刻通知工廠人員
  • 預防保養
    • 提供完整的機台歷史數據給RMA與支援單位作為參考
    • 若數據趨勢顯示出高風險，會立即發出警示
    • 藉由演算法分析大數據，能夠對機台狀況進行預測
  • 定位追蹤
    • 資產定位管理
    • 人員引導與追蹤
