Data Collection Station
for smart factory

DCS1

DCS1 is a Data Collection Station equipped with BLE 4.2, 32-bit MCU and various sensors for smart factory application.

4.2

Bluetooth/BLE Mesh

Increase speed and reliability of data transfer between BLE devices. Specifically for IoT application.
Also support BLE mesh network.

32-bit MCU

32-bit high performance MCU.

Various Sensors

Various built-in sensors tailored to smart factory application.

Feature List

  • Bluetooth low energy (BLE) 4.2
  • Support BLE Mesh
  • Temperature and humidity sensor
  • Pressure sensor
  • 3-axis accelerometer
  • RS-485 MODBUS sensor
  • 32-bit MCU up to 48MHz
  • Optional Wifi and LoRa connections
  • LCD Display: 144x64 Dot-Matrix LCD
  • Buttons: 6 Options
  • LED Indicators: 4 outputs
  • IO Definition
      • RS-485
      I2C
      Two USB 2.0 ports
      ST-Link Interface for MCU STM32F429
      Four AC outputs (AC Neutal Common)
      Four DC inputs
  • AC Power 100-240VAC, 0.12A
  • CE and FCC Certified
  • OS Support: Windows and Linux
  • Dimension: 150mm x 100mm x 98mm

Applications

IoT

  • Smart Factory
  • Temperature measurement
  • Humidity measurement
  • Machine vibration detection
