Wi-Fi SiP Module

M101

The MtM+ M101 is a Wi-Fi SiP module. It is an equipped with 802.11 b/g/n (1x1) Wi-Fi for your connectivity needs. Ideal for portable devices needing connectivity.

Size

9.2 x 8.3 x 1.1

Power

3.3v / 1.2v (Optional)

Package

LGA-52 pin

Interface

SDIO/GSPI

Feature List

  • Complies with 802.11 b/g/n
  • Supports 20/40 MHz bandwidth
  • Supports SDIO 1.1/2.0/3.0 up to 100MHz clock rate
  • Single 3.3v Power only
  • RoHS compliant / Lead Free
